Supported RobotsÔÉÅ

This page hosts a list of supported robots and references to them. To add your robot, submit a PR to this page on Github!

Official (supported by robot manufacturer):

  • Universal Robots

  • Franka Emika research robots

Unofficial (from the community):

  • KUKA industrial robots (KUKA Robot Sensor Interface (RSI))

  • KUKA IIWA (KUKA Fast Robot Interface (FRI))

  • KUKA LBR and IIWA (KUKA Fast Robot Interface (FRI))

  • ABB - EGM interface

  • Mitsubishi RV1A

End-effectors:

  • Schunk SVH 5-finger Hand

Non robot-devices:

  • Force Dimension haptic devices

  • ODrive Motor Controller

  • ODRI Motor Controller

Mobile manipulators:

  • Hello Robot - Stretch (for simple simulation with MoveIt2)
